Line Destroyer
by Belloah

What is Line Destroyer?

Line Destroyer is a tool design to remove line breaks generated when copying text from PDF to other places.

How to use Line Destroyer?
  • Pressing "Go banana!" can paste text from clipboard, transform it, display and copy the result to clipboard
  • If you would like to input text manually, uncheck "Need help in putting banana?"
  • If you need to copy the result to clipboard, check "Love minion?"
  • If you would like to run transform only, uncheck both
  • Green button at the bottom also copy content in result textarea to clipboard
    (If "Love minion?" is checked before pressing "Go banana!", the result is copyed to clipboard and no need to press it)
  • If you would like to input multiple paragraphs, check "Bananas?", which paragraphs are kept and 2 line breaks are added in between
  • Toggle "Bananas?" also clear paragraphs kept
